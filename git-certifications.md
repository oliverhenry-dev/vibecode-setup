# Git & GitHub — Free Courses + Certificates

> Tour reference handout — git & GitHub credentials you can earn alongside the tour.
> Most are free and issue a shareable certificate or badge. Earn one, then add it to LinkedIn.

---

## Top pick — GitHub Foundations Certification

Official GitHub credential — a real, verifiable badge covering exactly our workflow:
repos, branches, commits, pull requests, collaboration. The exam is normally paid (~US$99),
but students can often get it **free via GitHub Education** (student verification / voucher
campaigns — check eligibility on the page).

- Certification + student eligibility: https://education.github.com/experiences/foundations_certificate
- Free prep: 4-course Git & GitHub skill track on DataCamp (linked from the page above)

## LinkedIn Learning — git/GitHub paths (Premium)

Structured git / GitHub courses with completion certificates that post straight to your
LinkedIn profile. **Requires LinkedIn Premium** — but there's a **1-month free trial**,
enough to finish a path and claim the cert (cancel before billing).

- Browse: https://www.linkedin.com/learning/browse/certifications

## Recommended path for tour students

1. **Start free** — a free course with a completion cert (Codecademy / Alison / Great Learning).
2. **Go official** — **GitHub Foundations** if you can get the student-free exam (best credential).
3. **If you want more** — LinkedIn Learning git path on the free trial.
4. Add every badge to your LinkedIn profile.
